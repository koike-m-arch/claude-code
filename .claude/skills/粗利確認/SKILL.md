# 粗利確認スキル

## 目的

全体数値スプレッドシートの「月次×チーム【粗利】_新版」シートから以下の2つを取得・表示する。

1. **グロース1課の月次粗利**（チーム全体）
2. **小池勝也の個人粗利**（到達見込みベース）

## 対象スプレッドシート

- **ファイル名**: 全体数値
- **スプレッドシートID**: `1RYZdyu26lzbPF7EE0nYXFriSFK-stxbMNYQXw_FGPmw`
- **Googleアカウント**: `koike-m@mirai-kirei.jp`
- **対象シート**: `月次×チーム【粗利】_新版`

## シートの列構造

- 列A〜C: チームコード・名前等のラベル
- 列D〜F: 2025年5月（売上・原価・粗利）
- 以降、月ごとに3列ずつ追加（売上・原価・粗利）
- **最新月（当月）はシートの末尾3列**

## データ取得方法

`mcp__google-workspace__read_sheet_values` を1回呼び出す。

- **範囲**: `月次×チーム【粗利】_新版!A1:AM110`

この1回の読み取りで以下の両方が取得できる：

| データ | 行 | 列（4月の場合） |
|--------|----|----------------|
| グロース1課（チーム全体） | **行16** | AM列（粗利） |
| 小池勝也（個人・到達見込み） | **行101** | AM列（粗利） |

- 行1〜2でヘッダー（月）を確認し、対象月の粗利列を特定する
- 月を指定されない場合は当月の列を使用する

## 粗利目標（月次更新）

**設定月: 2026年4月**

| 対象 | 月次目標粗利 |
|------|-------------|
| グロース1課（チーム） | ¥13,500,000 |
| 小池勝也（個人） | ¥5,700,000 |

### 月替わりチェック（必須）

スキル実行の最初に、**設定月と当月を必ず比較する**。

- **一致する場合** → そのまま上記の目標値を使用して粗利を表示する
- **一致しない場合（月が替わっている）** → 粗利データを取得する前に以下を確認する：

```
今月（○月）の目標粗利を教えてください。
グロース1課（チーム）目標：
個人目標：
```

ユーザーから回答を受け取ったら：
1. このスキルファイルの「設定月」と「月次目標粗利」を新しい値に書き換える（Editツール使用）
2. その後、粗利データを取得して表示する

## 出力フォーマット

達成率 = 実績粗利 ÷ 目標粗利 × 100（小数点第1位まで）

```
グロース1課：¥XX,XXX,XXX/¥5,700,000　XX.X%
個人　　　  ：¥X,XXX,XXX/¥1,264,445　XX.X%
```

このフォーマットで2行のみ出力する。
