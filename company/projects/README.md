# projects/ — 案件管理

進行中・完了した案件の情報を管理します。

## フォルダ構成例

```
projects/
├── README.md
├── 2026_clientA_SEO/        # 案件フォルダ（年_クライアント_種別）
│   ├── overview.md          # 案件概要・スケジュール
│   ├── tasks.md             # タスク一覧
│   ├── reports/             # レポート格納
│   └── assets/              # 素材・画像等
└── 2026_clientB_listing/
    └── ...
```

## overview.md テンプレート

```markdown
# 案件名

## 概要
- クライアント:
- 種別: SEO / リスティング / SNS / その他
- 開始日:
- 終了予定日:

## 目標
-

## 進捗
- [ ] タスク1
- [ ] タスク2
```
